# Frontend Portfolio - Issa Dovale

A modern, responsive portfolio website built with React, TypeScript, and TailwindCSS.

## Project Overview

This portfolio website showcases Issa Dovale's frontend and backend development skills, projects, and professional experience.

## Features

- Responsive design that works across devices
- Light/dark mode support
- Multi-language support (English/Spanish)
- Modern UI with smooth animations
- Project showcase with filterable categories
- Contact form with validation

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- shadcn/ui components
- React Router
- React Query

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to the project directory
cd dov-frontend-verse

# Install dependencies
npm install

# Start the development server
npm run dev
